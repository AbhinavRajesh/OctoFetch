## OctoFetch
A Nicer look at GitHub Profile

### Tech Stack Used
* Visual Studio Code
* JavaScript
* React.JS Framework
* GitHub API


### Installation

1. Clone the Repository
    ```shell
    git clone https://github.com/AbhinavRajesh/OctoFetch.git
    ```
    
2. Navigate to Downloaded file
    ```shell
    cd OctoFetch
    ```
3. Install required Dependencies
    ```shell
    npm install
    ```
4. Run the App!
    ```shell 
    npm start
    ```
